# Process Overview
Program Management reviews the new order, assigns resources, establishes project timelines, and conducts kickoff meeting with all stakeholders. This process ensures the entire team understands requirements and is aligned on deliverables and schedule.

## Responsibilities
- **Primary Department**: Program Management
- **Supporting Roles**: Engineering, Manufacturing, Planning
- **Key Stakeholders**: Sales, Customer, Engineering Leads, Manufacturing Leads

## Inputs
What this process receives to begin work:
- Sales Order from ERP system
- Customer PO and specifications
- Project number assignment
- Customer delivery requirements
- Any special customer requirements or contract terms

## Process Steps
1. **Review Order Details**: Analyze customer requirements, specifications, and delivery timeline
2. **Assign Project Manager**: Designate PM responsible for this project
3. **Create Project Plan**: Develop high-level schedule with key milestones
4. **Assemble Team**: Identify engineering, manufacturing, and support resources needed
5. **Schedule Kickoff Meeting**: Invite all stakeholders for project launch meeting
6. **Conduct Kickoff**: Review requirements, assign responsibilities, confirm schedule
7. **Document Decisions**: Record meeting notes and action items in project management system

## Outputs
What this process produces for the next stage:
- Project plan with timeline and milestones
- Resource assignments documented
- Kickoff meeting notes and action items
- Requirements passed to Engineering for design
- Manufacturing planning receives preliminary schedule
- Risk register initiated

## Tools & Systems
- Project Management software (MS Project, Jira, Monday.com, etc.)
- SharePoint project folder structure
- Meeting scheduling system
- ERP system for resource planning

## Notes
- Kickoff meetings should occur within 3 business days of order receipt
- Ensure customer is invited to kickoff if contract requires customer involvement
- Document any changes to standard process or special requirements in meeting notes
- If order involves new technology or capabilities, schedule additional technical review
- Export-controlled projects require security briefing during kickoff
